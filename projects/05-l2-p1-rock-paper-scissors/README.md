# Level 2 - Project 1: Stone Paper Scissors

## Description
A console-based implementation of the classic game. This project focuses on logic separation and using enums for game states.

## How to Run
1. Open the root `Engineering-Logic-Roadmap.slnx` or the local `05-Algorithms-L2-ProjectA.slnx`.
2. Set this project as the **Startup Project**.
3. Press `Ctrl + F5`.
